# wsdc
基于JavaWeb的网上订餐系统

## 项目介绍
整个项目使用 Servlet + JavaBean + 少量JSTL 完成，属于一个练习Servlet的小项目。

## 运行环境
Tomcat v7.0 + MySQL5

## 开发工具
Eclipse + MySQL-Front
